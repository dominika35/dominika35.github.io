# Kod-polex

## Potrzebne będą:

* [Node.js](http://nodejs.org/) (with NPM)
* [Bower](http://bower.io/)
* [Ember CLI](http://www.ember-cli.com/)

## Instalacja

* `npm install`
* `bower install`

## Uruchamianie

* `ember server`
* Aplikacja dostępna na: [http://localhost:4200](http://localhost:4200).
